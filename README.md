IsNumberEven. Python tutorials. Algorithms. Input/Output. Console Application. 
A console application in the Pascal programming language. 
The program does the following:
  1. Prompts input a natural number (an integer positive).
  2. Calculates a modulo to 2 of the number.
  3. if the modulo is equal 0 the number is even or odd in another case.

IsNumberEven. Занятия по Python. Алгоритмы. Простой ввод-вывод. Консольное приложение.
Консольное приложение написанное на языке программирования Pascal. 
Программа делает следующее:
  1. Просит ввести натуральное число (целое положительное).
  2. Вычисляет остаток от деления числа на 2.
  3. Если остаток равен 0, то число чётное и нечётное в противном случае.
Разработано в интегрированной среде PascalABCDotNet.
